# Changelog

All notable changes to the Supymem VS Code extension will be documented in this file.

## [0.1.0] - 2024-12-22

### Added
- Initial release
- **Ask Agent**: Query your team's knowledge base with natural language (`Cmd+Shift+K`)
- **Why Decision**: Get context on why code decisions were made
- **Who Knows**: Find team experts for specific files
- **Store Knowledge**: Save selected text to the knowledge base
- **View Decisions**: Browse and explore team decisions
- **My Tasks**: View and manage your assigned tasks

### Views
- Tasks sidebar panel with priority and status indicators
- Decisions sidebar panel with importance levels
- Team Activity feed with recent commits, PRs, and reviews

### Configuration
- `supymem.apiUrl`: Configure backend API URL
- `supymem.teamId`: Set your team identifier
- `supymem.username`: Set your username for activity tracking
